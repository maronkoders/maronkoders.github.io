My Nhaka Dashboard

My Nhaka Dashboard, is a simple platform intended to automate business processes for Nhaka Life Assurance. This platform has modules like business writing, claims, and accounting.

On the development of this software, I was the key frontend developer. I used Laravel together with Vue.js to consume web services provided on Swagger Online rest API client.My main responsibilities on this project spun from Simple CRUD operations to development of complex multiple data capturing pages that required the use of Vue.js components. Just to give a brief list of major tasks I performed:

  • Implemented routes security protocols,
  • Implemented live validation rules through the system,
  • Designed generic functions that consumed Web services.
